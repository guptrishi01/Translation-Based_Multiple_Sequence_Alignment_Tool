Script of Choice: Detect Open Reading Frame

Start codon: {ATG}
Stop codon: {TAA, TAG, TGA}

Description: The script will identify ORFs from a FASTA file by detecting a start DNA read and an end DNA read and adding that sequence into a list. The function will take in a FASTA file and output a list of DNA reads.

Input:
*FASTA: String
- The script will take in a FASTA file as a string (i.e. "sequence.fasta")

Output:
*ORFs: List[SeqRecord]
- The script will output a list of ORFs obtained from DNA sequence

The script will have a shebang line at the beginning to indicate what interpreter to use in executing it.

The script will have a function header with appropriate function and input names. The script's function headers will also utilize type hints for error-handling and informative purposes.

The script will specify the DNA start codons and stop codons in two separate lists. The script will also create a Boolean check variable and an empty list for ORFs.

The script will load the BioPython package and utilize the SeqIO.parse() function to read in the FASTA file and creates an iterator. The script will make a list out of the iterator).

The script will contain a helper function that finds a start codon and a corresponding end codon. The function will use the start codon and end codon and create a SeqRecord, and will repeat this process until all ORFs of the input DNA sequence are found. This function will output a list entry of SeqRecords.

The script will use the iterator and go through each record and obtain the raw DNA sequence; with that raw DNA sequence, use the ORF helper function and append the ORFs found to the ORF list.

The script will return the list of SeqRecords.
